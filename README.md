# sokol-gen
A sokol project generator

## Usage
1. Clone the repo
2. `go build -o ./bin/sokol-gen`
3. Add bin to path
4. sokol-gen anywhere to start generating a sokol project

## Features
- Makefile
- main.c
- html shell, css for wasm
- sokol headers at will
